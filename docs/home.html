Real~Currents




This site is a showcase of experiments in Information Experience Design (IxD).

The emphasis of each post is on different ways to process and prepare data with the R programming language, while using the Svelte reactive web app framework to present interactive insights into the informative implications of that data.

The foundational methods of this technological synthesis (or tech stack) have been distilled into SveltR, a template for interactive data analysis and visualization.
